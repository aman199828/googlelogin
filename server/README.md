
# Google Login Demo




## Run Locally

Clone the project

```bash
  git clone https://github.com/aman199828/googlelogin.git
```


# First select server folder 

Install dependencies

```bash
npm i

```

Start the server

```bash
node index.js
```

# Then select googlelogin folder inside server folder 

Install dependencies

```bash
npm i

```

Start the server

```bash
npm run dev
```
